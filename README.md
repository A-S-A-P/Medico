## Front-end: Put in /templates folder


## Django Install Instructions



Linux/Mac:
```
pip install pipenv 
```

Windows:
```
python -m pip install pipenv
```


```
pipenv --python 3.6 install django==2.2 // Replace python 3.6 with your version of python (python3 --version / py --version)

pipenv shell

```
